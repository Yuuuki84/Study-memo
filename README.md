# Study Memo

**Study Memo** は、学習・調査内容を整理・記録するためのシンプルな Web メモアプリです。  
ローカルファースト設計をベースに、必要に応じて **Firebase (Google ログイン)** を用いたクラウド同期を行うことができます。

> Latest commit: **v10-2**

---

## ✨ 特徴

- 📝 **学習メモの一元管理**  
  タイトル / URL / タグ / 要点 / 追記メモを 1 件ずつ整理して記録できます。

- 🔍 **検索・並び替え**  
  タイトルやタグを使って素早く目的のメモを見つけられます。

- ⭐ **Star / Trash 管理**  
  重要なメモの強調表示や、ゴミ箱による安全な削除が可能です。

- ☁ **クラウド同期（任意）**  
  Google ログイン後、自分の UID に紐づいてデータを Firestore に保存できます。

- 👀 **ゲスト閲覧対応**  
  未ログインでも、公開設定されたデータ（`publicLogs/default`）を閲覧できます。

- 🧩 **local-first 設計**  
  通常操作はすべてローカルストレージで完結し、同期は明示的な操作時のみ実行されます。

---

## 🚀 使い方

### 1. ゲスト（未ログイン）
- 公開データを閲覧できます。
- 編集・保存・同期はできません。

### 2. ログインユーザー
- 右上の **「ログイン」** ボタンから Google ログインします。
- 自分専用のメモを作成・編集できます。
- ☁ **クラウド同期** ボタンで、任意のタイミングで保存・復元が可能です。

## 📂 データ構成（概要）

- **localStorage**
  - 学習メモ（Active / Trash）
  - 同期状態・最終同期時刻

- **Firestore**
  - `learnLogs/{uid}` : ログインユーザー専用データ
  - `publicLogs/default` : ゲスト向け公開データ

---

## 🔐 セキュリティ・設計方針

- 認証なしではクラウド書き込み不可
- ゲストは閲覧専用
- 同期・復元は必ずユーザー操作をトリガーに実行
