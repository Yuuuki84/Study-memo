# 学習記録ログ（Learning Log）

日々の調査内容・学習メモを **ローカル保存（localStorage）を主軸**に記録し、  
必要に応じて **Firebase（Firestore）へ自動バックアップ**できるシンプルな学習ログアプリです。

UI は「書く・見る・追記する」に集中できる構成を重視しています。

---

## 特徴

- ✅ **local-first 設計**
  - 通常利用はすべてブラウザの `localStorage`
  - オフラインでも利用可能
- ✅ **Firebase は裏側でバックアップ**
  - 匿名認証（UIなし）
  - 失敗しても操作感に影響しません
- ✅ **編集機能あり**
  - 一覧クリック → 入力フォームに内容を再表示
  - 「追加」ボタンで **新規 / 更新を自動判別**
- ✅ **改行（行間）を完全保持**
  - 要点・メモの改行がそのまま表示／再編集可能
- ✅ **エクスポート / インポート**
  - JSON 形式でデータを持ち運び可能

---

## 画面構成

- **左ペイン**
  - 検索
  - 新規記録 / 編集フォーム
  - 一覧
- **右ペイン**
  - 選択中の記録の詳細表示
  - 追記メモ

---

## 主な機能

### 記録の追加
1. タイトル（必須）を入力
2. URL / タグ / 要点（任意）を入力
3. **「追加」** をクリック

---

### 記録の編集
1. 一覧から項目をクリック
2. 左の入力欄に内容が自動反映
3. 修正後 **「追加」** をクリック  
   → 新規作成ではなく **更新** されます

---

### 追記メモ
- 選択中の記録に対して複数メモを追加可能
- 改行を含む文章もそのまま保持

---

### 検索
- タイトル / URL / タグ / 要点 / メモ全文から検索
- 入力と同時にリアルタイム反映

---

### エクスポート / インポート
- **エクスポート**
  - JSON ファイルとして保存
- **インポート**
  - 既存データをまとめて読み込み

---

## データ構造（概要）

```json
{
  "id": "uuid",
  "title": "タイトル",
  "url": "https://example.com",
  "tags": ["tag1", "tag2"],
  "summary": "要点（改行あり）",
  "notes": [
    {
      "id": "uuid",
      "text": "追記メモ（改行あり）",
      "at": 1710000000000
    }
  ],
  "createdAt": 1710000000000,
  "updatedAt": 1710000000000
}
``
